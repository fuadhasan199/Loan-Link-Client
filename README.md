 <h2>  Loan-Link | Micro-Loan Management System
Loan-Link is a comprehensive Micro-Loan management platform designed to streamline the loan application and approval process. It features a role-based dashboard system for Borrowers, Managers, and Admins to interact seamlessly. </h2>

                                                            Live Demo : fuad-bank.pages.dev  







<h1>Features :  </h1> 
Secure Authentication: Integrated with Firebase for Google Social Login and Email-based authentication.

Dynamic Catalog: Browse all available loan types with real-time data fetched from MongoDB.

Interactive Forms: Advanced loan application forms built with React Hook Form for smooth validation and UX.

Role-Based Dashboard: Separate layouts and access permissions for Admin, Manager, and Borrower.

Real-time Status Tracking: Borrowers can track their loan status (Pending/Approved/Rejected) instantly.

Responsive UI: Fully optimized for Mobile, Tablet, and Desktop using Tailwind CSS and DaisyUI.

Data Integrity: Efficient data handling using Axios for all CRUD operations. 


Backend: Node.js, Express.js

Database: MongoDB Atlas

Authentication: Firebase (Google Sign-in, Email/Password)

State Management & Data Fetching: React Hooks, Axios

Form Handling: React Hook Form

Routing: React Router  






<h1> 
Role-Based Functionalities : </h1>
                                <h3> Borrower :</h3>
View available loan packages.

Apply for loans with a detailed application form.

View personalized "My Loans" page to track application status.

Pay application fees (Unpaid/Paid status).

<h3>  Loan Officer (Manager) :</h3>
Access a dashboard to view all pending applications.

Review user details and financial information.

Approve or Reject loan applications.

  <h3>Admin: </h3>
Manage all registered users.

Update user roles (Promote to Manager/Admin).

Control homepage content (Toggle which loans appear on the home page).
